# Patrick's Knowledge Base

> A version-controlled repository of my learning, ideas, and experience.

---

# Purpose

This repository is my personal knowledge base. Its purpose is to capture, organize, and continuously improve what I learn throughout my career.

Rather than storing isolated summaries of books and courses, this repository is designed to become a living body of knowledge that reflects my own understanding.

Topics include:

* Artificial Intelligence
* Machine Learning
* Data Engineering
* Software Engineering
* Statistics
* Economics
* Supply Chain
* Leadership
* Personal Projects

---

# Guiding Principles

## Learn Actively

Learning starts with reading, experimentation, and handwritten notes. AI is used to organize and refine knowledge—not replace the learning process.

## One Concept, One Note

Each concept should have a single canonical note.

As I revisit topics through additional books, courses, or projects, I update the existing note rather than creating duplicates.

## Continuous Improvement

Notes are living documents.

They should be periodically refactored to:

* Improve explanations
* Remove duplication
* Add examples
* Create links between related concepts
* Reflect new understanding

## Practical First

Whenever possible, connect concepts to:

* Real work
* Personal projects
* Teaching
* Content creation
* Future ideas
